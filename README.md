# Sistem-Login-Multi---User
Sistem Login Multi-User adalah sistem yang dirancang untuk mensimulasikan atau mengimplementasikan mekanisme login pada sistem operasi yang mendukung banyak pengguna (multi-user). Sistem ini memungkinkan beberapa pengguna memiliki akun terpisah dengan hak akses dan direktori masing-masing, seperti pada sistem operasi Linux.
